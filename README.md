Data Visualization, Preprocessing and Statistical Analysis

## Purpose
The purpose of this lab is to explore a dataset using data visualization, data preprocessing, and statistical analysis techniques in Python using Jupyter Notebook.

## Dataset
The "tips" dataset was used, which contains information about restaurant bills, tips, customer demographics, and dining details.

## Key Insights
- Scatter plots show a positive relationship between total bill and tip amount.
- Histogram shows most bills fall within a moderate price range.
- Bar charts reveal differences in average bills across different days.
- Correlation analysis indicates strong relationships between total bill and tip.

## Data Preprocessing
The dataset was cleaned by checking for missing values, detecting and removing outliers using the IQR method, performing sampling, reducing dimensions by dropping columns, applying Min-Max scaling, and discretizing continuous values.

## Statistical Analysis
Statistical techniques including central tendency measures, dispersion measures, and correlation analysis were applied to better understand the dataset.
